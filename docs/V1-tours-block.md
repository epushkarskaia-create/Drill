# V1 — Блок туров (сохранённая версия)

Состояние на момент сохранения: вкладка **Tours** в правой панели с выбором и запуском туров.

## Текст и структура

- **Заголовок/описание:** «Start a guided tour to explore the rig.»
- **Тур 1:** Value & Advantages Tour — 3–5 min — кнопка «Start tour»
- **Тур 2:** How to Start / Pre-start Checks — 5–7 min — кнопка «Start tour»

## Поведение

- При выборе тура вызывается `startTour("valueTour")` или `startTour("startupTour")`, устанавливается первый POI тура, вкладка переключалась на Tours.
- Дальше отображался список шагов тура, прогресс, кнопки Next/Prev/Exit и диалог выхода.

## Где лежал код

- Список туров (экран выбора): `src/components/panel/TourRunner.tsx` — блок `if (!tour)` (строки ~55–88).
- Полный раннер тура (шаги, прогресс, кнопки): тот же файл `TourRunner.tsx`.

Компонент `TourRunner` оставлен в проекте (файл не удалён); при необходимости блок выбора туров можно снова вынести в отдельную вкладку или виджет, используя этот файл как референс.
